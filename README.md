# 📚 NoteGenius AI — Smart Lecture Summarizer

> **Hackathon Project**  
> Convert lectures, articles, or PDFs into summaries, key points, flashcards, and quizzes using AI.

NoteGenius AI helps students **learn faster and revise efficiently** by automatically transforming long educational content into structured study material.

---

# ⚡ Technology Stack

| Layer | Technology |
|------|-------------|
| Frontend | React + TypeScript |
| Build Tool | Vite |
| AI Model | LLaMA 3.3 70B |
| AI Platform | Groq API |
| Storage | Browser LocalStorage |
| Package Manager | npm |
| Version Control | Git & GitHub |

---

# 🚀 Features

### 📄 AI Lecture Summarization
Convert long lectures or articles into **short and easy-to-read summaries**.

### 📌 Key Points Extraction
Automatically generates **important bullet points** from the content.

### 🧠 Flashcards Generator
Creates **interactive flashcards** for quick revision and memory retention.

### ❓ Quiz Generator
Generates **multiple choice questions (MCQs)** to test understanding.

### 📂 File Upload Support
Users can upload **documents or notes** and generate study materials instantly.

### 💾 Save Notes
Generated notes are stored in **browser localStorage** so users can access them later.

### ⚡ Fast AI Processing
Uses **Groq inference engine** with **LLaMA 3.3 70B** for ultra-fast responses.

### 🎨 Student-Friendly UI
Simple and clean interface designed for easy learning.
